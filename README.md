# MySellComb

Web Crawling Dashboard for Automated Data Collection

---

## Overview

MySellComb은 Playwright와 Flask를 활용하여 웹 데이터를 자동으로 수집하고 관리하기 위해 개발한 개인 프로젝트입니다.

반복적인 데이터 수집 업무를 자동화하고 수집한 데이터를 Dashboard를 통해 확인할 수 있도록 구성하였습니다.

---

## Features

- Web Crawling Automation
- Playwright 기반 브라우저 자동화
- Flask Dashboard
- 데이터 저장 및 관리
- 반복 작업 자동화

---

## Architecture

```
Target Website
        │
        ▼
 Playwright
        │
        ▼
 Data Processing
        │
        ▼
 Google Sheets / Database
        │
        ▼
 Flask Dashboard
```

---

## Why I Built This

반복적인 데이터 수집 업무를 줄이고 데이터를 자동으로 수집·저장하기 위해 개발했습니다.

크롤링부터 저장, 조회까지 하나의 흐름으로 구현하여 데이터 자동화 경험을 쌓는 것을 목표로 했습니다.
